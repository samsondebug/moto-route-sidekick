Setup instructions (30 seconds)
  1. Go to supabase.com and create a free project.
  2. Settings → API: copy Project URL and anon public key.
  3. SQL Editor → run: create table positions (room text, handle text, lat float8, lng float8, ts bigint, primary key(room,handle)); alter table positions enable row level security; create policy "public rw" on positions for all using (true) with check (true); alter publication supabase_realtime add table positions;
  4. Paste URL + key above, hit Save.

Data

Everything (bike, rides, routes, settings, badges, photos) lives on this device. Nothing is sent to a server. Shared routes and Ride DNA cards are encoded client-side.
